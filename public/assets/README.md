# Assets Directory

## Logo File
Place your `opnpage_logo.png` file in this directory.

The logo should be:
- PNG format with transparent background
- Approximately 200-300px wide for optimal display
- Navy (#0F2233) color scheme to match the brand

The application expects the logo at:
```
/assets/opnpage_logo.png
```

This path is referenced in:
- Header component: [Header.jsx](../../src/components/Header.jsx)
- Footer component: [Footer.jsx](../../src/components/Footer.jsx)
